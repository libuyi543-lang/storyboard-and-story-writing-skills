# Storyboard Story Toolkit

Reusable workflows, checklists, prompts, and examples for two connected creative tasks:

- storyboard design
- narrative writing

This toolkit is tool-agnostic. Humans can read it directly, and AI assistants can use it as structured context.

## What It Contains

- `storyboard/`
  distilled principles, workflow, review checklist, and course map for shot design and storyboard planning
- `story-writing/`
  distilled principles, workflow, scene/dialogue guidance, review checklist, and course map for story and script development
- `prompts/`
  reusable prompt templates for turning the toolkit into practical outputs
- `examples/`
  small example outputs that show the expected format
- `adapters/`
  lightweight instructions for using the toolkit with Codex, Claude, or ChatGPT

## Suggested Use

### If you are a human creator

1. Start in `story-writing/` if you only have an idea, premise, or partial plot.
2. Start in `storyboard/` if the story already exists and you need shot planning.
3. Use the checklist files last, during revision.

### If you are using an AI assistant

1. Give the assistant the task.
2. Point it to the relevant folder.
3. Ask it to follow the workflow, not just summarize the files.
4. Ask for a concrete output format such as beat sheet, scene list, shot list, or revision notes.

## Minimal File Guide

- Story development:
  `story-writing/principles.md`
  `story-writing/workflow.md`
  `story-writing/scene-and-dialogue.md`
  `story-writing/checklist.md`

- Storyboard development:
  `storyboard/principles.md`
  `storyboard/workflow.md`
  `storyboard/checklist.md`

## Good Default Outputs

- story writing:
  `logline + central question + three-layer stakes + beat sheet + scene list`
- storyboard:
  `Shot # | Beat | Purpose | Viewpoint | Framing/Size | Height/Angle | Composition/Focus | Blocking/Movement | Cut In/Out Logic`

## Origin

This toolkit was distilled from course notes and PDF summaries, then reorganized into reusable documents.

# Course Map

This file maps the distilled skill rules back to the provided course material.

## Theory

- `分镜课程-01-05课`
  - shot-reverse-shot as baseline
  - axis and five-position grammar
  - perspective basics

- `分镜课程-06-08课`
  - main shot / 主镜
  - derive camera from image
  - compress dialogue into beat-level story units

- `分镜课程-09-10课`
  - six principles for shot size
  - six principles for camera height
  - avoid same-size same-composition cuts
  - avoid drawer-pulling
  - use gradual tightening/loosening

- `分镜课程-11-12课`
  - two ways to think about composition
  - beauty plus meaning
  - visual focus, space, and frame rhythm

- `分镜课程-13-14课`
  - spatial depth
  - camera/character movement

- `分镜课程-15-16课`
  - when to break axis
  - viewpoint as audience alignment

- `分镜课程-17课`
  - montage as meaning created by juxtaposition

## Practice

- `分镜实战-01-04课`
  - start from a default coverage idea, then refine
  - use establishing relation shots before dialogue coverage

- `分镜实战-05-08课`
  - images must tell story, not merely illustrate nouns
  - every decision still answers the same question: does it tell the story better?

- `分镜实战-09-10课`
  - isolate one technical focus while still serving story

- `分镜实战-11-12课`
  - viewpoint is created by combining what a character sees and how they react

- `分镜实战-13-16课`
  - staging and blocking introduce new camera problems and opportunities
  - rhythm, axis, and viewpoint may compete; choose consciously

- `分镜实战-17-18课`
  - integrated application across a whole scene
  - combine previous rules rather than treating them as separate tricks

# Codex Adapter

Use these prompts when working with Codex:

- Story writing:
  `Use the toolkit in /path/to/storyboard-story-toolkit/story-writing to turn this premise into a beat sheet and scene list.`

- Storyboard:
  `Use the toolkit in /path/to/storyboard-story-toolkit/storyboard to turn this scene into a shot-by-shot storyboard.`

Codex works best when you also specify:

- target format
- length
- genre
- whether you want diagnosis, rewrite, or clean output

# ChatGPT Adapter

Use these prompts when working with ChatGPT:

- Story writing:
  `Use the story-writing toolkit below as the method guide. Build this idea into a complete story outline with central question, three-layer stakes, five core beats, and a scene list.`

- Storyboard:
  `Use the storyboard toolkit below as the method guide. Turn this scene into a storyboard shot list with shot purpose, viewpoint, size, angle, composition, movement, and cut logic.`

ChatGPT usually benefits from:

- pasting only the relevant files, not the whole toolkit at once
- asking for one stage at a time
- requesting a revision pass with the checklist after the first draft

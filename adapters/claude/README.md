# Claude Adapter

Use these prompts when working with Claude:

- Story writing:
  `Please use the workflow and principles in the attached story-writing folder to develop this premise into a logline, central question, beat sheet, and scene list.`

- Storyboard:
  `Please use the workflow and principles in the attached storyboard folder to convert this scene into a shot list with viewpoint, framing, blocking, and cut logic.`

Claude usually benefits from:

- attached reference files or pasted excerpts
- explicit output headings
- one clear task per prompt

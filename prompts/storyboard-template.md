# Storyboard Prompt Template

Use this template with any AI assistant:

```text
Use the files in `storyboard/` as the primary method guide.

Task:
[Describe the script, scene, or beat sequence to storyboard.]

Requirements:
- Follow `storyboard/workflow.md`.
- Keep the board story-first: each shot must clarify information, emotion, rhythm, relationship, or viewpoint.
- Use `storyboard/principles.md` for shot decisions.
- If revising an existing board, also use `storyboard/checklist.md`.

Output format:
Shot # | Beat | Purpose | Viewpoint | Framing/Size | Height/Angle | Composition/Focus | Blocking/Movement | Cut In/Out Logic
```
